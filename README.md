## Quick Start

1. Install [Ollama](https://ollama.ai) and pull a model: `ollama pull llama3`
2. Buy a tool on [Gumroad](https://nexusmind30.gumroad.com)
3. Unzip, install deps: `pip install -r requirements.txt`
4. Run the script. Done.
